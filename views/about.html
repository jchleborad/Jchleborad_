About Me and this Site

My Project Page contains links to the projects that I have created while working on my Web Developer Certification from Woz U and others as I continue to learn and grow in my web development knowledge.

The Robot Store and Car Dealership where developed as part of the curriculum, and include HTML5, CSS3, JavaScript and jQuery. CSS Less and Bootstrap were used in the styling of the pages.

Create, Design, Organize! was my final Project and was written using AngularJS. Bootstrap 4 is used for most of the styling

Cindy & Me, the travel blog, is a site I began working on after completion of the course and receiving my SWD Certification. It is written using AngularJS and Bootstrap 4 to make the site responsive and friendly to mobile devices. This is an ongoing project and is in development constantly.

I will add to the list as I create new projects. Hope you enjoy!
